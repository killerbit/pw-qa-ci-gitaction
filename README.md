# 🚀 Playwright Automation Framework

A scalable UI Test Automation Framework built using Playwright + TypeScript following the Page Object Model (POM) design pattern with Ortoni HTML Reporting.

This framework is designed for maintainable, reusable, and scalable test automation suitable for enterprise CI/CD environments.

---

# 📌 Tech Stack

Automation Tool: Microsoft Playwright  
Language: TypeScript  
Test Runner: Playwright Test Runner  
Design Pattern: Page Object Model (POM)  
Reporting: Ortoni HTML Report  
Version Control: Git + GitHub  
CI/CD Ready: Jenkins / GitHub Actions
